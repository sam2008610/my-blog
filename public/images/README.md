# 封面圖片說明

## 如何添加封面圖片

1. **網站預設封面**：將圖片命名為 `default-cover.jpg` 並放在此目錄
2. **文章封面**：將圖片放在此目錄，然後在文章的 front matter 中指定：
   ```yaml
   images: ["/images/your-image.jpg"]
   ```

## 建議的圖片規格

- **尺寸**：建議寬度至少 1200px
- **格式**：JPG、PNG、WebP
- **檔案大小**：建議小於 500KB 以確保載入速度

## 範例

- `default-cover.jpg` - 網站首頁預設封面
- `post-cover.jpg` - 文章封面範例
